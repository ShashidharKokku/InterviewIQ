# InterviewIQ

InterviewIQ is a simple AI simulated web application that helps students prepare for interviews by generating questions and answers based on skills or input.

## Features
- Skill-based interview questions
- Simple UI
- Built with Flask, HTML, CSS

## Tech Stack
- Python (Flask)
- HTML
- CSS

## How to Run
1. Download or clone this repository
2. Install Python
3. Install Flask
4. Run: python app.py
5. Open browser and go to http://127.0.0.1:5000
